Privacy Policy

Last Updated: June 5, 2026

1. Introduction

This Privacy Policy explains what information doggo the helper ("the Bot") collects, stores, and uses when you interact with the Bot through Discord.

2. Information We Collect

To provide its functionality, the Bot may collect and store:

  • Discord User IDs
  • Discord Channel IDs
  • Discord Server IDs
  • Bot-related configuration data
  • Reminder content created by users, including text and scheduled delivery time
  • Giveaway participation data (such as user IDs and entry records)

The Bot does not intentionally collect additional personal information beyond what is necessary for its features to operate.

3. Feature-Based Data Collection

The Bot may store data depending on which features are used. This includes but is not limited to reminders, giveaways, leaderboards, and other future features that require persistent storage.

New features added in the future may store similar operational data as needed for their functionality. Only data required for the feature to work is stored.

4. Message Content

The Bot does not store the contents of normal Discord messages.

Message content is only stored when a user intentionally provides it to a Bot feature that requires storage, such as reminders or similar commands.

5. Open Source Transparency

doggo the helper is open source. This means the bot’s source code, including updates and modifications, is publicly available for review.

Anyone can inspect how the Bot processes and stores data at any time.

6. How Information Is Used

  • Delivering reminders
  • Managing giveaway entries and selections
  • Saving server configuration
  • Operating commands requiring persistent data
  • Providing public features such as leaderboards

7. Data Visibility

Some stored data may be visible through public commands (for example, leaderboard or giveaway-related commands).

Database access is restricted to the bot owner and authorized maintainers.

8. Data Retention

Data is retained until it is manually deleted from the database or is no longer required for bot operation.

9. Third-Party Services

The Bot does not use third-party analytics services and does not sell or share data.

The Bot does not use AI services to process user content.

10. Data Security

Reasonable measures are taken to protect stored data. However, no system is completely secure.

11. Data Deletion Requests

Users may request deletion of their data by contacting the bot owner via Discord direct message.

12. Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of the Bot constitutes acceptance of the updated policy.

13. Contact

Questions about this Privacy Policy can be directed to the bot owner via Discord direct message.
